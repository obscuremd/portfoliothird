## Overview

I created a personal portfolio to showcase my development projects and skills. The portfolio is designed with a strong focus on aesthetics and usability, using ShadCN UI and Acerernity UI libraries to build a visually stunning and highly responsive interface. The project highlights my ability to craft clean, modular code with React and Next.js, delivering an engaging experience across devices.

## Key Features

- **Modern Design Libraries**: Leveraged ShadCN UI and Acerernity UI to build a cohesive and elegant design system.
- **Responsive Layout**: Ensured the portfolio is fully responsive and accessible on all devices and screen sizes.
- **Clean Component Architecture**: Built with React and Next.js for reusable, maintainable components and fast performance.
- **Easy Deployment**: Hosted on Vercel for seamless continuous deployment and quick updates.

## Technologies Used

- **Figma**: For designing polished, user-friendly UI/UX prototypes.
- **React and Next.js**: For building a performant, SEO-friendly frontend with reusable components and server-side rendering.
- **ShadCN UI & Acerernity UI**: For implementing a consistent, modern design system with scalable styling.
- **Vercel**: For easy deployment and automatic CI/CD integration, enabling rapid iteration.

## Challenges and Learnings

One challenge was balancing visual appeal with performance and responsiveness. I focused on optimizing component structure and styling to ensure smooth navigation and quick load times, while maintaining a striking and cohesive design.

## Outcome

The portfolio successfully showcases my projects and skills with a seamless, engaging user experience. It reflects my growth as a developer and designer, combining technical expertise with strong visual storytelling.
